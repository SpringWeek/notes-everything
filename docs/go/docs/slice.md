# go语言切片

* 切片本身没有数据，切片是对底层数据的一个view  
* 改变切面数据内容的值，底层数据的值也会改变   
* slice可以嵌套多层使用  
```go 
s := arr[2:6]
s = s[:3]
s = s[1:]
```
