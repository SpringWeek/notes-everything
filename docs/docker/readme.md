# docker

## docker 是什么？

1. 轻量级的操作系统虚拟化解决方案，能够在Linux系统上迅速创建一个容器(类似于 虚拟机)并在容器上部署和运行应用程序，同时还可以通过配置文件轻松将应用程序 的自动化安装、部署和升级。 
2. 基于Linux容器(LXC)技术，利用namespace的隔离机制隔离资源和cgoup的限额 功能进行资源管理。
